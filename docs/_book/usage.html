2 用建表文件创建

准备一个建表文件table_meta.txt

name : "test"
ttl: 144000
ttl_type : "kAbsoluteTime"
partition_num: 8
replica_num: 3
column_desc {
  name : "card"
  type : "string"
  add_ts_idx : true
}
column_desc {
  name : "mcc"
  type : "string"
  add_ts_idx : true
}
column_desc {
  name : "amt"
  type : "float"
  add_ts_idx : false
}

> put test 1534992888000 card0 mcc1 12.3

> put test 1534992910000 card1 mcc1 10.7
